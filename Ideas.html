1) SmartDoorCamera:

Problem Statement:

We hardly see any solutions which provides smart camera near your door. CCTV records the video, it does not have intelligence to detect/recognise human near the door.

Applications:

  • Smart Homes
  • Remote Security Management

Domains:

IoT, Sensors, LTE, Storage, Android/iOS

Solution:

Need to provide a camera+computing device (human motion detection and face recognition)+SIM.

Features :

  • Detect people when near to door
  • Audio/ Video Call owner using SIM
  • Send MAIL with picture when call miss.
  • Keep a recording until specifically deleted.
  • Two way communication.
  • Controls like door locks.
  • Security to prevent hacking.
  • An app to be used on mobile, laptops

References:

 

Implementation:

RPi+Sensors+Face detection algorithm+camera+SIM module+Installation
